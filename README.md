# 📥 Download Progress Bar

An interactive download progress bar built using HTML, Tailwind CSS, and JavaScript.

## 🚀 Features

- Animated progress bar
- Real-time percentage update
- Button state change (Download → Downloaded)
- Prevents multiple clicks

## 🛠 Tech Used

- HTML
- Tailwind CSS
- JavaScript (setInterval, DOM manipulation)

## ▶️ How to Run

- Open `index.html` in your browser

## 🧠 What I Learned

- setTimeout vs setInterval
- DOM manipulation
- Handling user clicks
- Updating UI dynamically
